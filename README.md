# Data-Lake-Anomaly-Detection-Pipeline
O projeto consiste no desenvolvimento de um pipeline de dados em datalake voltado ao monitoramento de índices relacionados a margem de contribuição, com o objetivo de integrar bases financeiras da organização e utilizar técnicas de machine learning para automatizar a detecção de anomalias
